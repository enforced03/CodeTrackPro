# 🚀 CodeTrack Pro

CodeTrack Pro is a Python-based GUI application that helps students track their coding practice and analyze their performance.

## 📌 Features
- Add coding problems
- View stored entries
- Difficulty analysis
- Weak topic detection
- Coding streak calculation

## 🛠 Technologies Used
- Python
- Tkinter
- JSON File Handling
- Dictionaries, Sets, Functions

## ▶️ How to Run
1. Download files
2. Run:
   python project.py

## 🎯 Purpose
To help students improve coding skills through tracking and analytics.
